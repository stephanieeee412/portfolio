GenLinkSG — SkillSwap Platform

Flask SQLite Bootstrap Jinja2

A youth–senior skill exchange web app where users list skills they can teach or learn, explore listings, and request a swap — designed to be simple, friendly, and mobile responsive.

Role: Full-stack (Flask + DB + UI)
Audience: Youth & seniors (intergenerational)
Focus: Usability, simple flow, clean visuals

The Highlights

Listings + Filters: browse by skill category, mode (teach/learn), and keyword search.
Request Flow: users can submit swap requests and track request status.
CRUD Dashboard: add/edit/delete listings with validation and clean UI.
Accessibility: larger buttons, clear contrast, and simple content structure for mixed-age users.

What I Built

  • Flask routes + templates (Jinja2) for listings, detail pages, and request submission.
  • SQLite schema for users, listings, requests, and categories.
  • Bootstrap UI with pastel theme, responsive layout, and reusable components.
